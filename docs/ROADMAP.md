## Product Principles

Our roadmap follows these principles:

1. **User-First:** Features are driven by user needs, not trends
2. **Simplicity:** Easy to use, hard to misuse
3. **Transparency:** Open roadmap, honest communication
4. **Privacy:** GDPR compliance, EU hosting, no data selling
5. **Fairness:** Cryptographically secure draws, verifiable results
6. **Speed:** Fast development cycles, ship early, iterate
